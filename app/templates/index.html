How This Service Ships

  • CI gates: pytest · Bandit (SAST) · pip-audit (SCA) · Trivy (container scan)
  • Infrastructure as Code: AWS CDK (Python) — network, IAM, ECR, ECS, logging stacks
  • Policy as Code: cdk-nag (AwsSolutions pack) on every CDK synth
  • Governance: protected main, PR reviews, approval-gated releases
  • Runtime: ECS Fargate · ALB with ACM TLS · CloudWatch observability

Project

This application exists to demonstrate a secure software delivery lifecycle end-to-end, implemented twice — once in Terraform, once in AWS CDK — so both approaches can be compared directly. The full source, pipelines, and stack code are public.
